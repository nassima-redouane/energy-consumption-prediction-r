# 🔋 Energy Consumption Prediction & Model Comparison (R)

## 📊 Project Overview
This project focuses on analyzing and predicting energy consumption using advanced regression techniques in R. 
The objective is to understand how different variables influence energy usage and to compare multiple statistical models to identify the most effective approach.

---

## 🎯 Objectives
- Analyze energy consumption patterns  
- Identify key factors influencing energy usage  
- Build and compare multiple regression models  
- Evaluate model performance and reliability  

---

## 🛠️ Tools & Technologies
- R  
- RStudio  
- ggplot2  
- dplyr  
- Statistical modeling (Linear, Polynomial, Interaction Models)  

---

## 📈 Methodology

### 1. Data Preparation
- Data cleaning and preprocessing  
- Handling missing values and inconsistencies  
- Data validation to ensure quality  

### 2. Exploratory Data Analysis (EDA)
- Visualization of relationships between variables  
- Identification of trends and patterns  

### 3. Model Development
Three types of regression models were implemented:

- **Multiple Linear Regression** → baseline model  
- **Polynomial Regression** → captures non-linear relationships  
- **Interaction Model** → evaluates combined effects between variables  

### 4. Model Evaluation
Models were compared using:
- R² (coefficient of determination)  
- ANOVA  
- Residual analysis  

---

## 📊 Results & Insights
- The **polynomial model improved performance** compared to the basic linear model  
- **Interaction terms showed limited statistical significance**  
- The final model was selected based on a balance between **performance and interpretability**
